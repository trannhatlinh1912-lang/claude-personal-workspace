     Kế hoạch: Personal Workspace với Claude Code Skills

     Context

     Xây dựng không gian làm việc cá nhân có cấu trúc rõ ràng, bao gồm 3 custom
     skills (slash commands), code trên GitHub, output files từ mỗi skill, và
     log lịch sử trò chuyện tự động.

     ---
     Kiến trúc 3 lớp

     ~/personal-workspace/      ← Git repo trên GitHub
         skills/
             research-reviews/  ← symlink ← ~/.claude/skills/research-reviews
             github-sync/       ← symlink ← ~/.claude/skills/github-sync
             log-session/       ← symlink ← ~/.claude/skills/log-session
         outputs/               ← Files output từ mỗi skill run
         logs/                  ← YYYY-MM-DD.md (auto-log qua Stop hook)
         hooks/
             stop-hook.sh

     ~/.claude/skills/          ← Claude Code đọc từ đây (symlinks tới repo)
     ~/.claude/settings.json    ← Thêm Stop hook config
     ~/.claude/serpapi.env      ← API key (NGOÀI repo, không commit)

     ---
     3 Skills

     1. /research-reviews — SerpAPI Google Maps (external API + files)

     Files: skill.md, fetch-reviews.py, report-template.md
     Output: outputs/research-reviews/YYYY-MM-DD_{place_name}_report.md

     Flow:
     1. Đọc SERPAPI_KEY từ ~/.claude/serpapi.env
     2. Chạy fetch-reviews.py <Google Maps URL> → JSON
     3. Lưu raw JSON + tạo báo cáo Markdown theo report-template.md
     4. Print đường dẫn output

     2. /github-sync — GitHub CLI / git push (CLI + external platform)

     Files: skill.md, sync.sh

     Flow:
     1. git status trong ~/personal-workspace — nếu sạch thì báo "Nothing to
     sync"
     2. git add outputs/ logs/ skills/ → commit với timestamp
     3. git push origin main
     4. Print danh sách files và commit hash

     3. /log-session — Ghi chép thủ công (files + folders)

     Files: skill.md (pure prompt skill, không cần script)

     Flow:
     1. Xác định log file: logs/YYYY-MM-DD.md
     2. Tóm tắt session hiện tại → format có cấu trúc (Đã làm / Kết quả / Tiếp
     theo)
     3. Append vào file, tạo header nếu chưa có

     ---
     Auto-logging (Stop Hook)

     Sau mỗi response của Claude, stop-hook.sh tự động append vào
     logs/YYYY-MM-DD.md:
     - Timestamp, session_id, working directory
     - Snippet cuối của assistant message (≤300 ký tự)
     - Luôn exit 0 — không bao giờ block Claude

     Config trong ~/.claude/settings.json:
     "hooks": {
       "Stop": [{
         "hooks": [{
           "type": "command",
           "command": "bash 
     /Users/trannhatlinh/personal-workspace/hooks/stop-hook.sh",
           "timeout": 10
         }]
       }]
     }

     Bảo mật SerpAPI key

     - Lưu tại ~/.claude/serpapi.env (chmod 600)
     - Nằm ngoài repo ~/personal-workspace/ → không bao giờ bị commit lên GitHub
     - fetch-reviews.py đọc bằng absolute path Path.home() / ".claude" / 
     "serpapi.env"
